for the Miami-Dade County .

Learn about and build apps with civic data for MIAMI.

This project is open source! Help build this directory of tools & resources!

Contribute to miamidata.org by forking the repository on GitHub. If you're not about to fork the repository and just want to suggest a data source, app, or other resource, you can currently make that suggestion via our issues queue on GitHub.

Apps

Explore apps built with open civic data.

    Transportation

  • BusDrone – See the position of Seattle busses in real time! Source on GitHub
  • PDXLiveBus – See the position of TriMet busses in real time! Source on GitHub
  • VeloRacks – Find bike racks and nearby businesses.
  • MetroCuts - See how Seattle metro funding will affect routes. Source on GitHub
  • wherecanipark.in - Interpret Seattle Street Parking to determine where you can park. Source on Web App and API on GitHub

    Educational / How To

  • What Do I Do With...? – Allows residents and businesses in King County to find locations to recycle and responsibly dispose of unwanted items. Source on GitHub

    Fire / Crime

  • @seattle911 twitter bot – Source on GitHub

    Food safety / restaurant inspections

  • Dinegerous – King County Food Inspections

    Maps

  • Seattle's Heritage Trees – Code on GitHub Source data
  • Map My Districts – Determine if a set of addresses are within a neighborhood or district boundaries by uploading bulk spreadsheets of Seattle addresses or drawing arbitrary polygons. Currently an internal tool for the Seattle Chamber of Commerce but may open to the public in the future.

    Wikis

  • SeattleWiki – A community-authored resource for Seattle, WA
  • PortlandWiki – The community powered knowledge commons for Portland, Oregon.
  • localwiki.net/pdx – A community-authored resource for Portland, Oregon.
  • BoiseWiki – A community-authored resource for Boise, Idaho
  • OlyWiki – A community-authored resource for Olympia, WA
  • Port Townsend Wiki – A community-authored resource for Port Townsend, WA
  • There are more Pacific Northwest wikis on localwiki.net, and you can start a new one there, too!
